# Templates - 模板库

> 各种笔记模板，快速创建标准化笔记。

## 可用模板

| 模板 | 用途 |
|------|------|
| `daily-template.md` | 每日笔记 |
| `article-note.md` | 文章笔记 |
| `book-note.md` | 书籍笔记 |
| `podcast-note.md` | 播客笔记 |
| `project-note.md` | 项目笔记 |
| `concept-card.md` | 概念卡片 |

## 使用方法

在 Obsidian 中使用 Templater 或 QuickAdd 插件调用模板。
