# Daily - 每日笔记

> 记录每天的想法、日程、反思。

## 文件命名规范

```
YYYY-MM-DD.md  (例如:2026-04-10.md)
```

## 模板

```markdown
# {{date}} Daily

## 今日完成
- 

## 思考
- 

## 明日待办
- [ ] 

## 捕获箱
- 
```

## 自动任务

- 每天 9:00 自动抓取 AI 新闻（hubtoday-fetch.ps1）
- AI 新闻会追加到当日笔记的 `## AI 资讯` 部分

## 内容规则

- 使用中文编写（除专业术语外）
- 保持简洁
