# 全栈开发流程技能

## 三部分整合

### 第一部分：代码开发与代理调度
- coding-agent 调度规范
- 并行任务执行
- 进度同步规则

### 第二部分：UI 开发规范
- 页面结构规范
- 搜索栏规范
- 表格规范
- 弹窗表单规范

### 第三部分：接口字段对齐检查
- 五层架构体系（DB → Model → Controller → API → Page）
- P0/P1/P2 问题分级

## 开发流程

```
需求分析 → 编码代理执行 → 接口对齐检查 → 交付完成
```

## 质量检查清单

- [ ] 理解需求，明确接口契约
- [ ] 遵循 UI 布局规范
- [ ] 执行接口对齐检查
- [ ] P0 问题全部修复
