# Resources - 资源库

> 文章、播客、视频、书籍的原始笔记。

## 目录结构

```
03_Resources/
├── 00_Articles/      # 文章笔记
├── 01_Podcasts/      # 播客笔记
├── 02_Videos/        # 视频笔记
├── 03_Books/         # 书籍笔记
└── 04_Tweets/        # 社交媒体摘录
```

## 来源追踪

每条笔记应包含:
- **来源 URL**:原始链接
- **捕获日期**:何时保存
- **主题标签**:#tag

## 处理流程

1. 快速捕获 → Inbox
2. 初步处理 → 添加来源和标签
3. AI 编译 → 生成永久笔记到 /04_Knowledge
4. 归档 → Resources 作为原始参考

## 内容规则

- **标题**:保留英文（论文名等）
- **正文**:使用中文编写
- **标签**:中文 + 英文混合
