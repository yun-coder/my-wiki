# Knowledge - 永久知识库 ⚡

> **Karpathy LLM 知识库核心** — 由 AI 编译和维护的永久笔记。
> 
> 这是传统 RAG/向量数据库的替代方案：不是检索后生成，而是**边走边编译**。

## 核心理念

```
Sources (文章/播客/视频)
       ↓ 捕获
Inbox  → Resources (原始笔记)
       ↓ AI 处理 + 人工审核
Knowledge (永久笔记，AI 编译)
       ↓ 持续迭代
Memory (系统提示词的一部分)
```

## 目录结构

```
04_Knowledge/
├── 00_Cards/         # 原子笔记 - 最小知识单元
├── 01_Concepts/      # 概念 - 理解一个主题
├── 02_Projects/      # 项目知识 - 编译的项目相关知识
├── 03_Domains/       # 领域知识 - 专业领域积累
└── 99_Memory.md      # AI 记忆 - 供系统提示词使用 ⭐
```

## 卡片格式示例

```markdown
# {{概念名称}}

## 核心定义
> 一句话概括

## 关键要点
- **点1**：
- **点2**：

## 来源
- [原始文章](url)

## 标签
#tag1 #tag2

## 相关卡片
- [[相关卡片名]]
```

## AI 编译流程

当 AI 处理新信息时：
1. 识别核心概念
2. 编译成原子卡片
3. 更新 Memory 文件
4. 建立链接关系

## 内容规则

- 使用中文编写（除专业术语外）
- 保持笔记的原子性
- 定期回顾和整合
