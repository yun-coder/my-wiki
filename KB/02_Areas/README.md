# Areas - 领域

> 你正在追踪的重要领域：项目、目标、责任、承诺。

## 目录结构

```
02_Areas/
├── 00_Projects/      # 项目 - 有明确终点的工作
├── 01_Goals/         # 目标 - 长期/短期目标
├── 02_Health/         # 健康 - 身体、心理健康追踪
├── 03_Career/        # 职业发展
└── 04_Relationships/ # 人际关系
```

## 核心原则

- 每个 Area 应该有明确的责任人和时间范围
- 项目有清晰的目标和完成标准
- 定期回顾（建议每周）
